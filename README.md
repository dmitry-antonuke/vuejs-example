# Startup the Server

```bash
cd server
node server.js
```

# Startup the Client

```bash
cd client
vue ui
```

* Go to the browser.
* Import the Project if necessary.
* On the left sidebar click `Tasks`.
* Click `serve` in the list of opened project tasks.
* Click `Run task` button.
* Wait for the task running.
* Click `Open app` button.
* Enjoy.
